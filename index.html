Install Bridge

One command. No account, no sign-up. macOS (Apple Silicon).

$ curl -fsSL https://github.com/AkshobyaRaoSWE/bridge/releases/latest/download/install.sh | bash
or download the app · all releases
Use the command above — it installs and opens with no security prompt. Self-contained (bundles its own runtime); auto-updates after first launch.
Downloaded with a browser instead?
macOS will block it once with "can't verify… free of malware." On recent macOS, right-click → Open no longer works. Do one of these:
  1. Move Bridge to your Applications folder, then run:
    xattr -dr com.apple.quarantine /Applications/Bridge.app
    and open it. (easiest, always works)
  2. Or: double-click Bridge → click Done → open System Settings → Privacy & Security → scroll down → click Open Anyway next to Bridge → confirm.
